## PROMPT 5 — Flask Routes / Controller
Used to: Write the Flask Blueprint with all cart API endpoints

"Create a Flask Blueprint for the Cart System with URL prefix /cart.
Include these routes:
- GET /cart/ → render cart.html template
- GET /cart/api → return cart JSON
- POST /cart/add → validate and add item (validate: product_id is int, name not empty, price >= 0, qty in [1,99])
- POST /cart/update → update quantity
- POST /cart/remove → remove item
- POST /cart/clear → clear all items
All routes must check session.user_id and return 401 if not logged in.
Return appropriate HTTP status codes (400 for validation, 404 for not found)."
